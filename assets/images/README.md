This directory holds all image assets for the Konecta website.

Naming convention:

- hero-\*.jpg — Hero section imagery
- event-\*.jpg — Event photography (CIO Konect, Municipal Indaba, WiTechnology)
- team-\*.jpg — Leadership / team headshots
- project-\*.jpg — Case study / impact project photos
- partner-\*.svg — Partner logos (prefer SVG)
- icon-\*.svg — UI icons (prefer SVG)
